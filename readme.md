## Installation

Clone repo then run:

```sh
make install
```

(it will back up your current .vim, .vimrc, and .gvimrc)
Note: If you have a Mac, you need a terminal that supports more colors like iTerm.

## Updating

To update plugins run:

```sh
make update
```
